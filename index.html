About This Page

Ini adalah demonstrasi kustomisasi halaman JavaScript sederhana. Pake tombol kontrol diatas untuk merubah tampilannya ya!

How It Works

Setelah kalian pilih perubahan tampilannya, maka halaman akan mengikuti sesuai apa yang kalian pilih. Hal ini dilakukan dengan memodifikasi kelas CSS dan gaya sebaris.

Features Implemented

  • Ubah warna latar belakang menggunakan dropdown
  • Menambah atau mengurangi ukuran font menggunakan tombol
  • Beralih antara mode terang dan gelap
  • Ubah gaya font (sans-serif, serif, atau monospace)
